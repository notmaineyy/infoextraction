Information Extraction Tools



About the Project

This website combines 3 different tools that are used to extract and analyse text from PDF documents such as news articles. The tools used are Named-Entity Recognition (NER), Term Frequency-Inverse Document Frequency (TF-IDF) and Dependency Parser.

NER

A model trained to extract and classify certain occurrences in a piece of text into pre-defined categories. These categories include person, place, organisation, country etc.

TF-IDF

Statistical measure that evaluates how relevant a word is to a document in a collection of documents.

Dependency Parser

The process of examining the dependencies between the phrases of a sentence in order to determine its grammatical structure. With the structure, it is then able to display the relationships between the words.
